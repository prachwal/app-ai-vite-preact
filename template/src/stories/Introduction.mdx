# System Designu

Witaj w systemie designu naszej aplikacji! Ten zbiór komponentów został stworzony z wykorzystaniem:

- **Preact** - szybka alternatywa dla React
- **TypeScript** - silne typowanie dla lepszej jakości kodu  
- **Tailwind CSS v4** - najnowszy framework CSS z utilites-first
- **Storybook** - dokumentacja i testowanie komponentów

## Filosofia Designu

Nasz system designu opiera się na kilku kluczowych zasadach:

### 🎨 Konsystentność
Wszystkie komponenty używają spójnej palety kolorów, typografii i przestrzeni.

### 🧩 Modularność  
Każdy komponent jest niezależny i może być używany w różnych kontekstach.

### ♿ Dostępność
Komponenty są zaprojektowane z myślą o dostępności (a11y) i wspierają czytniki ekranu.

### 📱 Responsywność
Wszystkie komponenty działają na różnych rozmiarach ekranów.

## Paleta Kolorów

Nasz system używa następujących kolorów głównych:

- **Primary (Niebieski)**: `bg-blue-500` - główne akcje
- **Secondary (Szary)**: `bg-gray-500` - akcje drugorzędne  
- **Success (Zielony)**: `bg-green-500` - potwierdzenia
- **Danger (Czerwony)**: `bg-red-500` - usuwanie/błędy

## Typografia

Używamy systemowych fontów dla lepszej wydajności:
- Nagłówki: `font-bold`
- Tekst: `font-normal` 
- Akcenty: `font-semibold`

## Rozmiary

Standardowe rozmiary komponentów:
- **Small**: `sm` - kompaktowe interfejsy
- **Medium**: `md` - domyślny rozmiar (zalecany)
- **Large**: `lg` - akcenty i Call-to-Action

## Jak Używać

1. **Przeglądaj komponenty** w menu po lewej stronie
2. **Testuj właściwości** używając kontrolek w panelu dolnym
3. **Kopiuj kod** z zakładki "Docs" każdego komponentu
4. **Sprawdzaj dostępność** w zakładce "Accessibility"
